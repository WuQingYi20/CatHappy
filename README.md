# 🐱 专业要饭猫陪读器

一个温暖可爱的番茄钟学习陪伴应用，专为N3备考设计。

## ✨ 功能特点

### 核心功能
- **番茄钟计时器**：25分钟专注学习 + 5分钟休息，渐变进度环
- **可爱的猫咪陪伴**：不同状态的动画效果，点击互动
- **温暖的鼓励语**：中日双语，贴心陪伴
- **学习数据统计**：记录每日学习进度和连续天数
- **里程碑庆祝**：完成3/5/10个番茄钟时的特殊动画
- **音效反馈**：可开关的提示音和互动音效
- **猫咪满足度**：可视化学习进度

### 🎌 N3日语学习功能（新增！）
- **📚 每日日语短语**：每天一句N3励志短语，带发音和翻译
- **☕ 休息时间学习卡片**：翻转卡片设计，轻松学习N3口语
- **🏆 日语成就系统**：解锁日语称号（初心者→努力家→達人）
- **💬 中日双语鼓励**：所有消息都加入日语元素
- **🐱 日语猫咪互动**：点击猫咪获得日语鼓励

> 详细功能说明请查看 [N3_FEATURES.md](./N3_FEATURES.md)


```bash

```


## 🎮 使用说明

1. **开始学习**：点击"开始"按钮启动25分钟番茄钟
2. **与猫咪互动**：点击猫咪会有可爱的反应和鼓励语
3. **查看统计**：页面底部显示今日学习数据和连续天数
4. **音效控制**：点击底部音效按钮可以开关声音

## 🏗️ 技术栈

- **React 18**：前端框架
- **Vite**：构建工具
- **Tailwind CSS**：样式框架
- **Framer Motion**：动画库
- **Lucide React**：图标库
- **LocalStorage**：数据持久化

## 🎨 自定义

### 修改番茄钟时长

在 `src/hooks/useTimer.js` 中修改默认时长：

```javascript
// 工作时长（秒）
setNewDuration(1500); // 25分钟

// 休息时长（秒）
setNewDuration(300);  // 5分钟
```

### 添加自定义鼓励语

在 `src/utils/messages.js` 中添加你的消息：

```javascript
export const messages = {
  start: [
    "你的鼓励语1",
    "你的鼓励语2",
    // ...
  ],
  // ...
}
```

### 自定义彩蛋

在 `messages.js` 中添加特殊事件的消息，比如合租时的梗和回忆。

## 💡 特色功能

### 智能时间检测

- 深夜学习（23:00-6:00）：显示关心消息
- 周末学习：鼓励劳逸结合
- 早晨学习：提示效率最高

### 连续学习天数

自动计算连续学习天数，达到7天时会有特殊庆祝。

### 猫咪满足度

根据今日完成的番茄钟数量，显示猫咪的"吃饱"程度。
