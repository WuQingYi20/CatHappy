# 🎌 N3日语学习功能说明

## 功能概览

在原有的番茄钟学习应用基础上，增加了丰富的N3日语学习元素和Happy激励功能，让学习过程更加有趣和实用。

## 新增功能

### 1. 📚 每日日语短语 (Daily Phrase)

**位置：** 页面顶部，猫咪上方

**功能特点：**
- 每天自动更换一句N3级别的励志短语
- 包含完整的日语、罗马音、中文和英文翻译
- 点击发音按钮可听日语发音（使用浏览器TTS）
- 可展开/收起翻译，先尝试自己理解
- 短语分类标签（励志、日常、鼓励等）

**短语示例：**
```
継続は力なり
Keizoku wa chikara nari
坚持就是力量
Continuity is strength
```

**学习建议：**
- 每天开始学习前先读一遍
- 尝试理解后再查看翻译
- 点击发音按钮跟读练习

---

### 2. ☕ 休息时间日语卡片 (Break Time Flashcard)

**位置：** 休息模式时自动显示

**功能特点：**
- 翻转卡片设计，点击可查看翻译
- 展示N3常用口语表达
- 包含使用场景说明
- 可刷新获取新短语
- 美观的渐变卡片动画

**短语类型：**
- 请求等待："ちょっと待ってください"
- 表示理解："分かりました"
- 询问意思："どういう意味ですか"
- 请求重复："もう一度言ってください"
- 等等...

**使用方法：**
1. 休息时卡片自动出现
2. 正面显示日语和罗马音
3. 点击翻转查看中文和使用场景
4. 点击刷新按钮换下一个短语

| 5个 | 頑張り屋 | Ganbariya | 拼命三郎 |
| 10个 | 達人 | Tatsujin | 达人 |
| 连续7天 | 継続の力 | Keizoku no Chikara | 坚持的力量 |

**展示效果：**

## 使用场景

### 典型学习流程

1. **开始学习**
   - 查看今日日语短语
   - 点击发音按钮练习
   - 点击"开始"按钮
   - 收到中日双语鼓励

2. **学习中**
   - 专注25分钟
   - 随时可点击猫咪互动
   - 猫咪用日语鼓励你

3. **完成番茄钟**
   - 收到日语祝贺消息
   - 可能解锁日语成就称号
   - 烟花庆祝动画

4. **休息时间**
   - 自动显示日语口语卡片
   - 点击翻转学习新短语
   - 轻松愉快地积累词汇

5. **持续学习**
   - 每天新的励志短语
   - 达成里程碑获得称号
   - 看到自己的进步
